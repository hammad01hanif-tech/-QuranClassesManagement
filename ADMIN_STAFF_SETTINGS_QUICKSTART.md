# ⚡ دليل سريع - 3 خطوات فقط

## 1️⃣ افتح admin-staff-settings.html

```
انقر بزر الماوس الأيمن → Open with Live Server
أو افتحه مباشرة في المتصفح
```

---

## 2️⃣ عدّل إعدادات المعلمين

```
لكل معلم:
- الراتب: 4000 ريال
- بداية الدوام: 30 دقيقة بعد العصر
- نهاية الدوام: 20 دقيقة بعد العشاء
- الخصميات: حسب الحاجة
```

**اضغط "حفظ الإعدادات"** ✅

---

## 3️⃣ ماذا حدث؟

```
✅ تم حفظ الإعدادات في:
   Firestore → staffSettings/{teacherId}

✅ لا حاجة لإنشاء Collection يدوياً!
   سيُنشأ تلقائياً عند أول حفظ
```

---

## ❓ ماذا لو لم يظهر معلمون؟

**أضف معلم في Firestore أولاً:**

```javascript
Firebase Console → Firestore → classes → أضف مستند

{
  classId: "TCH001",
  teacherName: "محمد أحمد",
  role: "teacher",
  active: true
}
```

**ثم حدّث الصفحة** (F5)

---

## 🎯 النتيجة

```
الآن:
- كل معلم له إعدادات خاصة محفوظة في Firestore
- teacher.js سيجلب هذه الإعدادات (الخطوة التالية)
- الخصميات ستُحسب بناءً على الإعدادات
```
