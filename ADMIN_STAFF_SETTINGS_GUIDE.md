## ❓ الأسئلة الشائعة

### ❓ هل أحتاج إنشاء staffSettings يدوياً في Firebase Console؟

**الجواب:** ❌ **لا!**

عند أول حفظ، Firestore سيُنشئ:
- ✅ Collection: `staffSettings`
- ✅ Document: `{teacherId}`
- ✅ جميع الحقول تلقائياً

**لا تفعل شيء في Firebase Console!** 

---

### ❓ ماذا لو لم يظهر أي معلم؟

**الحلول:**

1. **تأكد من وجود معلمين في Firestore:**
   - افتح Firebase Console
   - Firestore Database → `classes`
   - ابحث عن مستندات بها: `role: "teacher"`

2. **إذا لم يوجد معلمون:**
   - أضف معلم في `classes` collection:
   ```javascript
   {
     classId: "TCH001",
     teacherName: "محمد أحمد",
     role: "teacher",
     active: true
   }
   ```

3. **حدّث الصفحة** (F5)

---

### ❓ ماذا لو ظهر خطأ في التحميل؟

**الأخطاء الشائعة:**

| الخطأ | السبب | الحل |
|-------|-------|------|
| "Permission denied" | قواعد Firestore | راجع Firebase Rules |
| "No teachers found" | لا معلمين في classes | أضف معلمين أولاً |
| "Network error" | لا إنترنت | تحقق من الاتصال |

---

### ❓ كيف أعدّل إعدادات معلم بعد حفظها؟

**ببساطة:**
1. افتح `admin-staff-settings.html` مرة أخرى
2. ستظهر الإعدادات المحفوظة
3. البطاقة ستقول: **"✅ مُعدّ"**
4. عدّل القيم
5. اضغط "حفظ" مرة أخرى
6. ✅ سيتم التحديث

---

### ❓ هل يمكنني تعديل إعدادات عدة معلمين؟

**نعم!** ✅
- كل معلم له نموذج منفصل
- يمكنك تعديل وحفظ كل واحد على حدة
- أو تعديل الجميع ثم حفظ واحد تلو الآخر

## 🎯 بعد الحفظ

### ماذا سيحدث؟

```
1. الإعدادات محفوظة في Firestore ✅
2. المعلم يفتح صفحته (teacher section)
3. النظام يجلب إعداداته من staffSettings
4. يحسب أوقات الدوام بناءً عليها
5. عند Check-out، يحسب الخصميات بناءً عليها
```

## 🔧 في حالة وجود مشكلة

### خطأ: "Cannot use import statement"

**الحل:** ✅ تم إصلاحه في النسخة الحالية

### خطأ: "Firebase not initialized"

**الحل:** 
- تأكد من تحميل Firebase SDK
- الصفحة الآن تُهيئ Firebase تلقائياً

### خطأ: "Permission denied"

**الحل:**
```javascript
// Firebase Rules
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /staffSettings/{staffId} {
      allow read, write: if request.auth != null;
    }
  }
}
```
