# 📄 دليل تحميل تقرير PDF للسجل الشهري

## ✨ الميزات

تم تطوير نظام متكامل لتحميل السجل الشهري كملف PDF بتصميم احترافي ومُحسّن.

### 🎯 المميزات الرئيسية

1. **جدول كامل للحضور**
   - عرض جميع أيام الشهر بالتفصيل
   - 8 أعمدة: اليوم، بداية الدوام، الحضور، خصم التأخير، نهاية الدوام، الانصراف، خصم الخروج المبكر، الملاحظات
   - إزالة رمز التعديل (✏️) من التاريخ

2. **تمييز بصري واضح**
   - 🔴 **غياب**: خلفية حمراء فاتحة
   - 🟢 **إجازة**: خلفية خضراء فاتحة
   - ⚪ **لم يتم التسجيل**: خلفية رمادية فاتحة
   - 🟡 **خصومات مسموحة**: خلفية برتقالية فاتحة
   - 🔥 **خصومات نشطة**: خلفية حمراء فاتحة + نص عريض

3. **قسم الملخص النهائي**
   - الراتب الأساسي
   - عدد الغيابات
   - خصومات الغياب
   - خصومات التأخير
   - خصومات الخروج المبكر
   - إجمالي الخصومات (مُميز بالأحمر)
   - الراتب المتوقع (مُميز بالأخضر)

4. **تحسينات التنسيق**
   - توجيه A4 أفقي (Landscape) لملائمة الجدول الكبير
   - تصميم أبيض نظيف ومريح للعين
   - عناوين واضحة ومُنسّقة
   - تذييل يحتوي على تاريخ الإنشاء ورقم الصفحة
   - تحجيم تلقائي للخطوط لضمان ملائمة صفحة واحدة

---

## 🚀 طريقة الاستخدام

### الخطوات:

1. افتح **لوحة معلومات المعلم** (Teacher Dashboard)
2. اضغط على **"عرض السجل الشهري"** لفتح نافذة الحضور
3. اختر الشهر المطلوب من القائمة المنسدلة
4. اضغط على زر **"تحميل PDF"** 📄
5. سيتم تحميل الملف بصيغة: `attendance-{teacherId}-{monthName}-{year}.pdf`

---

## 📊 محتوى التقرير

### 🗂️ الجدول الرئيسي

| العمود | الوصف |
|--------|-------|
| اليوم | يوم الأسبوع + التاريخ الميلادي |
| بداية | وقت بداية الدوام المحدد |
| حضور | وقت الحضور الفعلي (أو "إجازة"/"لم يتم التسجيل") |
| خصم تأخير | قيمة خصم التأخير أو "سماح" إذا تم الإعفاء |
| نهاية | وقت نهاية الدوام المحدد |
| انصراف | وقت الانصراف الفعلي |
| خصم خروج | قيمة خصم الخروج المبكر أو "سماح" إذا تم الإعفاء |
| ملاحظات | أي ملاحظات إضافية (إن وجدت) |

### 📋 قسم الملخص

يعرض ملخصًا ماليًا كاملًا بتنسيق 4 أعمدة (2 بيانات × 2 قيم):

| البيان | القيمة | البيان | القيمة |
|--------|--------|--------|--------|
| الراتب الأساسي | xxx SAR | عدد الغيابات | xx يوم |
| خصم الغياب | xxx SAR | خصم التأخير | xxx SAR |
| خصم الخروج | xxx SAR | **إجمالي الخصومات** | **xxx SAR** ⚠️ |
| - | - | **الراتب المتوقع** | **xxx SAR** ✅ |

## 🎨 التصميم

### الألوان المستخدمة:

- **رأس الجدول**: أزرق (#667EEA)
- **غياب**: أحمر فاتح (#FEE2E2)
- **إجازة**: أخضر فاتح (#DCFCE7)
- **لم يتم التسجيل**: رمادي فاتح (#F3F4F6)
- **خصم مسموح**: برتقالي فاتح (#FFEDD5) + نص أصفر داكن
- **إجمالي الخصومات**: أحمر فاتح (#FECACA)
- **الراتب المتوقع**: أخضر فاتح (#BBF7D0)

### الخطوط:

- العناوين: **Helvetica Bold** (حجم 16-11)
- الجدول الرئيسي: **Helvetica** (حجم 7-8)
- القسم الملخص: **Helvetica** (حجم 8-9)
- التذييل: **Helvetica** (حجم 7)

## ⚙️ التقنيات المستخدمة

- **html2canvas** v1.4.1: تحويل محتوى HTML إلى صورة (canvas)
- **jsPDF** v2.5.1: إنشاء ملفات PDF في المتصفح
- **Portrait Orientation**: توجيه عمودي A4 (210×297 مم)
- **High-Quality Rendering**: دقة عالية (scale: 2) لوضوح النص العربي

### 🌟 لماذا html2canvas؟

استخدام **html2canvas** يحل مشكلة اللغة العربية بشكل نهائي لأنه:
1. ✅ يحول HTML المُنسّق إلى صورة
2. ✅ يحفظ جميع التنسيقات والألوان بدقة
3. ✅ يدعم العربية بدون أي مشاكل في الترميز
4. ✅ يعرض النص كما تراه بالضبط في المتصفح

**ملاحظة**: هذه نفس الطريقة المستخدمة في تصدير تقارير الأجزاء القرآنية في قسم العارض.

## 🔧 المتطلبات التقنية

### المكتبات المطلوبة في `index.html`:

```html
<!-- html2canvas - لتحويل HTML إلى صورة -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/html2canvas/1.4.1/html2canvas.min.js"></script>

<!-- jsPDF Library - لإنشاء ملفات PDF -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js"></script>
```

**ملاحظة**: لا نحتاج jspdf-autotable لأننا نستخدم html2canvas بدلاً من الجداول النصية.

## 🐛 استكشاف الأخطاء

### المشكلة: النص العربي يظهر مشفر أو غير مفهوم
**الحل**: ✅ تم حل هذه المشكلة نهائيًا باستخدام html2canvas. النص يُعرض كصورة وليس كنص، لذا لن تظهر مشاكل الترميز.

### المشكلة: التقرير يستغرق وقتًا طويلاً
**الحل**: استخدام html2canvas يحتاج بضع ثوانٍ لتحويل HTML إلى صورة. هذا طبيعي وستظهر رسالة "جاري الإنشاء" أثناء العملية.

### المشكلة: لا يتم التحميل
**الحل**: تحقق من وجود مكتبات html2canvas و jsPDF في `index.html` وأنها محملة بشكل صحيح.
