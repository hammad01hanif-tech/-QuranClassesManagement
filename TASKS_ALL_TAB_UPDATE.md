# تحديث تبويبة "الكل" في نظام المهام

## نظرة عامة
تمت إضافة تبويبة جديدة "الكل" في بداية تبويبات المهام لعرض جميع المهام المضافة في مكان واحد، مع دعم كامل للفلترة السريعة بالمسؤول.

## المزايا الجديدة

### 1. تبويبة "الكل" (All Tasks)
- **الموقع**: في البداية قبل تبويبة "اليوم"
- **الوظيفة**: عرض جميع المهام بدون تصفية زمنية
- **المحتوى**: 
  - المهام النشطة (قيد الانتظار، جاري التنفيذ، متأخرة)
  - المهام المكتملة
  - المهام بتاريخ ومن دون تاريخ

### 2. الفلترة السريعة المتكاملة
- **التوافق**: تعمل الفلترة السريعة بالمسؤول مع تبويبة "الكل"
- **السلوك**:
  - اختيار "الكل" + "الكل" → عرض جميع المهام
  - اختيار "الكل" + "عبدالمجيد" → عرض مهام عبدالمجيد فقط
  - اختيار "الكل" + أي مسؤول → عرض مهام المسؤول المحدد

### 3. التبويبة الافتراضية
- **القديم**: تبويبة "اليوم" كانت الافتراضية
- **الجديد**: تبويبة "الكل" الآن هي الافتراضية
- **الفائدة**: عرض شامل عند فتح قسم المهام

## سيناريوهات الاستخدام

### السيناريو 1: عرض شامل
```
المستخدم يفتح قسم المهام
↓
يظهر تبويبة "الكل" نشطة تلقائياً
↓
عرض جميع المهام (20 مهمة مثلاً)
```

### السيناريو 2: فلترة بالمسؤول
```
المستخدم في تبويبة "الكل"
↓
ينقر على "عبدالمجيد" في الفلترة السريعة
↓
عرض مهام عبدالمجيد فقط (8 مهام مثلاً)
```

### السيناريو 3: التنقل بين التبويبات
```
المستخدم في "الكل" (20 مهمة)
↓
ينقر على "اليوم" (5 مهام)
↓
ينقر على "عبدالمجيد" (2 مهام من اليوم)
↓
يرجع لـ "الكل" (8 مهام لعبدالمجيد)
```

### السيناريو 4: إضافة مهمة جديدة
```
المستخدم يضيف مهمة جديدة
↓
يتم حفظ المهمة
↓
يرجع تلقائياً إلى تبويبة "الكل"
↓
يرى المهمة الجديدة مباشرة
```

### 1. تجربة مستخدم محسنة
