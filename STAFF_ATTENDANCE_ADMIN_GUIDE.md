# 📖 دليل قسم تحضير المعلمين والطاقم

## 🎯 الغرض من القسم
قسم إداري شامل لمتابعة واعتماد حضور وخصميات جميع أفراد الطاقم (معلمين، عارضين، إداريين) بتصميم احترافي متجاوب مع الأجهزة المحمولة.

---

## 🚀 الوصول للقسم

### من قسم "المزيد":
1. افتح قسم **المزيد** من القائمة السفلية
2. ابحث عن بطاقة **👨‍🏫 تحضير المعلمين والطاقم**
3. اضغط على البطاقة لفتح القسم

### الشكل العام:
```
┌─────────────────────────────────┐
│  👨‍🏫                          │
│  تحضير المعلمين والطاقم        │
│  متابعة واعتماد الحضور         │
│                            ←   │
└─────────────────────────────────┘
```

---

## 📱 واجهة القسم

### 1. **اختيار الموظف** (Staff Selector)
```
┌─────────────────────────────────┐
│  🧑‍🤝‍🧑 اختر الموظف           │
│  ▼ -- اختر من القائمة --        │
└─────────────────────────────────┘
```

**الخطوات:**
- انقر على القائمة المنسدلة
- ستظهر قائمة بجميع الطاقم مع أيقونات:
  * 👨‍🏫 معلم
  * 📊 عارض
  * 👨‍💼 إداري
- اختر الموظف المطلوب

---

### 2. **بطاقة معلومات الموظف** (Staff Info Card)
تظهر بعد اختيار الموظف:

```
┌─────────────────────────────────┐
│  👨‍🏫                          │
│  اسم المعلم                     │
│  معلم | 3000 ريال               │
└─────────────────────────────────┘
```

**تحتوي على:**
- أيقونة الدور
- الاسم الكامل
- نوع الوظيفة (معلم/عارض/إداري)
- الراتب الشهري

---

### 3. **فلترة حسب الشهر والسنة**
```
┌────────────┬────────────┬──────────┐
│ الشهر ▼   │ السنة ▼   │ [استعراض]│
└────────────┴────────────┴──────────┘
```

**الخطوات:**
1. اختر الشهر الهجري من القائمة
2. اختر السنة الهجرية
3. اضغط على زر **"استعراض التقرير"**

**الافتراضي:**
- الشهر الحالي (ذو القعدة)
- السنة الحالية (1448)
- يمكن الرجوع لآخر سنتين

---

## 📊 عرض جدول الحضور

### فتح التقرير:
```
┌─────────────────────────────────┐
│  📅 استعراض التقرير            │
│  عرض جدول الحضور الشهري        │
└─────────────────────────────────┘
```

بعد الضغط على "استعراض التقرير"، سيظهر نفس جدول الحضور الموجود في صفحة المعلم مع جميع التفاصيل:

| التاريخ | بدء الدوام | وقت الحضور | خصمية التأخير | نهاية الدوام | وقت الخروج | خصمية الخروج المبكر | الملاحظات |
|---------|------------|-------------|---------------|--------------|------------|---------------------|-----------|

## ⚡ نظام اعتماد الخصميات

### 🎯 القاعدة الأساسية:
> **جميع الخصميات تُعتمد تلقائياً بشكل افتراضي**

### 📱 كيفية التعديل (Mobile First):

#### 1. **النقر على الخصمية:**
- انقر على أي خلية تحتوي على خصمية (مبلغ بالريال)
- سيظهر **Bottom Sheet** من أسفل الشاشة

#### 2. **خيارات Bottom Sheet:**
```
┌─────────────────────────────────┐
│         خصمية التأخير           │
│─────────────────────────────────│
│  ✅ اعتماد الخصم               │
│  (يبقى الخصم كما هو)            │
│─────────────────────────────────│
│  🙏 سماح (إلغاء الخصم)         │
│  (يتم إلغاء الخصم نهائياً)      │
└─────────────────────────────────┘
```

#### 3. **الخيار الأول: اعتماد الخصم ✅**
- يبقى الخصم كما هو
- لون الخلية: **أحمر فاتح** (#fee2e2)
- الحالة: `approved`

#### 4. **الخيار الثاني: سماح 🙏**
- يتم إلغاء الخصم نهائياً
- لون الخلية: **أصفر فاتح** (#fef3c7)
- النص: ~~المبلغ~~ **سماح**
- الحالة: `pardoned`
- المبلغ يصبح **0 ريال**

---

## 🎨 الألوان والتنسيق

### 1. **الخصمية المعتمدة** (Approved)
```css
الخلفية: أحمر فاتح (#fee2e2)
النص: "50 ريال"
```

### 2. **الخصمية المسموح بها** (Pardoned)
```css
الخلفية: أصفر فاتح (#fef3c7)
النص: "~~50 ريال~~ سماح"
```

### 3. **لا يوجد خصم**
```css
النص: "—"
```

```javascript
  absenceApprovalStatus: 'approved' | 'pardoned',
  
  // معلومات التحديث:

  updatedBy: 'admin'

```

## 🔄 التحديث التلقائي

### عند السماح بخصمية:
1. **في صفحة الإدارة:**
   - تتحول الخلية للون الأصفر
   - يظهر نص "سماح"
   - الملخص يُحدّث تلقائياً

2. **في صفحة المعلم:**
   - عند فتح الجدول يرى التحديث مباشرة
   - الخصمية المسموح بها لا تُحسب في الملخص
   - الراتب الصافي يزيد بمقدار الخصمية الملغاة

---

## 📊 الملخص النهائي

### حساب الخصميات:
```javascript
// يتم استبعاد الخصميات المسموح بها:
totalDeductions = 
  (lateDeductions - pardonedLate) +
  (earlyLeaveDeductions - pardonedEarlyLeave) +
  (absenceDeductions - pardonedAbsence)
```

### مثال:
```
خصميات التأخير: 100 ريال
  - معتمدة: 50 ريال
  - مسموح بها: 50 ريال

خصميات الغياب: 200 ريال
  - معتمدة: 200 ريال

إجمالي الخصميات = 50 + 200 = 250 ريال
الراتب الصافي = 3000 - 250 = 2750 ريال
```

## 🎯 حالات الاستخدام

### السيناريو 1: معلم تأخر بعذر
```
1. الإداري يفتح قسم التحضير
2. يختار اسم المعلم
3. يستعرض التقرير
4. يرى خصمية التأخير (مثلاً 20 ريال)
5. ينقر على الخصمية
6. يختار "سماح" من Bottom Sheet
7. ✅ الخصمية تُلغى والملخص يُحدّث
```

### السيناريو 2: مراجعة شهر سابق
```
1. اختيار الموظف
2. اختيار الشهر السابق من الفلترة
3. استعراض التقرير
4. مراجعة جميع الخصميات
5. السماح بما يستحق السماح
```

### السيناريو 3: إعداد تقرير نهائي
```
1. فتح التقرير
2. مراجعة جميع الخصميات
3. اعتماد/سماح حسب الحاجة
4. الملخص النهائي يعكس القرارات
5. المعلم يرى الراتب الصافي الصحيح
```

## 📋 ملاحظات مهمة

### ⚠️ تنبيهات:
1. **الافتراضي:** جميع الخصميات معتمدة تلقائياً
2. **لا رجعة:** السماح نهائي (يمكن التراجع بإعادة الاعتماد)
3. **التاريخ:** يمكن الوصول للأشهر السابقة
4. **الصلاحية:** فقط الإداريون يمكنهم السماح

### ✅ أفضل الممارسات:
1. راجع جميع الخصميات قبل نهاية الشهر
2. تأكد من الأعذار قبل السماح
3. وثّق القرارات في الملاحظات
4. راجع الملخص النهائي قبل إغلاق الشهر
