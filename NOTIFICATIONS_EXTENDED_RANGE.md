# تمديد نطاق إشعارات الغياب 📋

## ملخص التحديث

تم تمديد نظام الإشعارات ليشمل **جميع الغيابات من الثاني وما بعده** بدلاً من الاقتصار على الثاني والثالث فقط.

---

## 🔄 التغييرات المطبقة

### 1. تعديل شرط الإشعار
**الملف:** `js/admin.js` - دالة `checkAndNotifyAbsenceViolations()`

**قبل:**
```javascript
if (absenceCount === 2 || absenceCount === 3) {
  // إنشاء إشعار فقط للثاني والثالث
}
```

**بعد:**
```javascript
if (absenceCount >= 2) {
  // إنشاء إشعار للثاني وما بعده (2، 3، 4، 5...)
}
```

### 2. إضافة نظام الأعداد الترتيبية العربية
**الملف:** `js/admin.js` - دالة `checkAndNotifyAbsenceViolations()`

**تم إضافة:**
```javascript
const ordinals = {
  2: 'ثاني', 
  3: 'ثالث', 
  4: 'رابع',
```

**النتيجة:**

- غياب 3 → "⚠️ غياب ثالث بدون عذر"

## ✅ السلوك الحالي

### الغياب الأول
- ❌ **لا يتم إنشاء إشعار**
- ⚠️ العقوبة: الوقوف خارج الحلقة (عقوبة داخلية)
- 💡 السبب: عقوبة بسيطة لا تحتاج إلى إشعار إداري

### الغياب الثاني
- ✅ **يتم إنشاء إشعار تلقائياً**
- 🚨 العقوبة: حرمان من دخول الحلقة واستدعاء ولي الأمر
- 🔔 الإشعار: "⚠️ غياب ثاني بدون عذر"

### الغياب الثالث
- ✅ **يتم إنشاء إشعار تلقائياً**
- ❌ العقوبة: استبعاد كامل من الحلقات (فصل)
- 🔔 الإشعار: "⚠️ غياب ثالث بدون عذر"

### الغياب الرابع وما بعده
- ✅ **يتم إنشاء إشعار تلقائياً لكل غياب**
- ❌ العقوبة: **نفس عقوبة الثالث** (استبعاد كامل/فصل)
- 🔔 الإشعار: "⚠️ غياب رابع/خامس/سادس... بدون عذر"
- 💡 الفائدة: التوثيق الكامل لجميع الغيابات للمتابعة الإدارية

## 🎯 الفوائد

1. **توثيق شامل:** يتم إنشاء سجل كامل لجميع الغيابات المتكررة
2. **متابعة أفضل:** المشرف يرى كل غياب جديد بدون عذر فور حدوثه
3. **تنبيه مستمر:** حتى بعد الفصل، يستمر النظام بالتوثيق (مفيد للإحصائيات)
4. **مرونة:** يمكن بسهولة تعديل الحد الأقصى للإشعارات إذا لزم الأمر

## 📝 ملاحظات تقنية

### العقوبات (كما هي)
```javascript
// في determineActionForAbsenceCount()
if (absenceCount === 1) return actions[1];  // وقوف
if (absenceCount === 2) return actions[2];  // حرمان واستدعاء
if (absenceCount >= 3) return actions[3];   // فصل (3 وما بعده)
```

**لم يتغير شيء في العقوبات!** فقط تم توسيع الإشعارات.

### الإشعارات (تغيرت)
```javascript
// في checkAndNotifyAbsenceViolations()
if (absenceCount >= 2) {  // كان: === 2 || === 3
  // إنشاء إشعار
}
```

### منع التكرار
```javascript
// كل إشعار له ID فريد:
const notificationId = `absence_${studentId}_${currentMonth}_${absenceCount}`;

// مثال: absence_STD123_1447-11_4
// إذا كان موجود، لا يتم إنشاؤه مرة أخرى
```
